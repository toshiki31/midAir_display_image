### 4.3.2 主観評価

7段階リッカート尺度による主観評価について、6つの評価項目それぞれに対してFriedman検定を実施した。その結果、**吹き出しの読みやすさ**において距離条件による非常に有意な効果が認められた（χ²(4) = 14.64, *p* = .006, Kendall's *W* = .46）。表2に読みやすさの記述統計量を示す。中央値は距離が近いほど高く（0 cm: 5.5、15 cm: 5.0、30 cm: 3.0、45 cm: 3.5、60 cm: 2.0）、近距離ほど読みやすいという明確な傾向が見られた。

6つの評価項目に対するBonferroni補正（α = .05 / 6 = .0083）を適用した場合でも、読みやすさは有意水準を満たした（*p* = .006 < .0083）。また、効果量は中程度（*W* = .46）であり、実質的にも統計的にも意味のある差と言える。

### 4.3.3 認知負荷

NASA-TLXによる認知負荷の評価に対してFriedman検定を実施した結果、距離条件による有意な差が認められた（χ²(4) = 11.79, *p* = .019, Kendall's *W* = .37）。表4に各距離条件における記述統計量を示す。中央値は0 cmで36.8、15 cmで35.7、30 cmで45.0、45 cmで43.3、60 cmで56.2であり、遠距離ほどワークロードが増加する傾向が見られた。

すべての条件でShapiro-Wilk検定により正規性が確認されたため（すべて *p* > .05）、パラメトリック検定である反復測定ANOVAも実施した。Mauchlyの球面性検定により球面性の仮定違反が検出されたため（*W* = .021, *p* = .016）、Greenhouse-Geisser補正を適用した結果、有意な主効果が確認された（*F*(1.78, 12.45) = 4.64, *p* = .035, ε = .45）。FriedmanとANOVAの両検定で一貫して有意な結果が得られたことから、距離条件がワークロードに及ぼす効果は頑健であると言える。

## 4.4 考察

### 4.4.1 仮説の検証

本研究では3つの仮説を設定したが、その検証結果は以下のとおりである。

**H1（タスクパフォーマンス）**については、距離条件による有意な差が認められ（*p* = .020, *W* = .37）、**支持された**。特に60 cm条件で顕著に遂行時間が長くなった（中央値: 0 cm = 22.3秒、60 cm = 27.4秒）。これは、遠距離では空中像の文字を読み取る際に視認性が低下し、正確な判別により時間を要したためと考えられる。4文字という短い刺激であっても、視距離が大きくなるとタスクパフォーマンスに測定可能な影響が生じることが示された。

**H2（主観評価）**については、読みやすさにおいて距離条件による非常に有意な効果が認められたため（*p* = .006, *W* = .46）、**強く支持された**。Bonferroni補正後も有意性を保ち、本研究で最も頑健な結果となった。中央値の傾向から、近距離ほど読みやすいという明確なパターンが確認された（0 cm: 5.5、60 cm: 2.0）。これは物理的な視距離と視認性の関係を反映しており、近距離ほど視角が大きく、文字が明瞭に見えるためと解釈できる。また、目の疲れ（*p* = .028）とぼやけ（*p* = .037）においても有意な傾向が見られ、遠距離条件が視覚的快適性に悪影響を及ぼすことが示唆された。

**H3（認知負荷）**については、距離条件による有意な差が認められ（*p* = .019, *W* = .37）、**支持された**。さらに正規性が確認されたため実施した反復測定ANOVAでも有意な主効果が確認され（Greenhouse-Geisser補正後 *p* = .035）、ノンパラメトリック検定とパラメトリック検定の両方で一貫した結果が得られた。遠距離ほどワークロードが増加する傾向が明確であり（中央値: 0 cm = 36.8、60 cm = 56.2）、空中像の視認性低下が主観的な作業負担の増加につながることが実証された。

### 4.4.2 距離効果の統合的解釈

本実験では、タスクパフォーマンス、主観評価、認知負荷の**すべての指標で一貫して距離効果が確認された**。これは、空中像インタフェースにおける飛び出し距離が、客観的なパフォーマンス、主観的な視認性、そして認知的ワークロードに系統的な影響を与えることを示す重要な知見である。

**読みやすさ**は最も強い効果を示し（*p* = .006, *W* = .46）、Bonferroni補正後も有意性を保った。中央値が示す単調減少傾向（近距離ほど高評価）は、視距離と視認性の物理的関係と一致する。空中像が顔に近いほど、網膜像が大きくなり、文字の細部がより明瞭に知覚される。本実験では文字サイズを150 pxに固定したため、距離が遠くなるほど視角が小さくなり、読みにくさが増したと考えられる。

**タスクパフォーマンス**と**認知負荷**でも同様の効果量（*W* = .37）が観察され、遠距離条件では遂行時間の増加とワークロードの上昇が生じた。これらの結果は相互に関連しており、視認性の低下が認知的努力の増大を招き、結果としてパフォーマンスの低下につながるという因果連鎖を示唆している。

### 4.4.3 実用的示唆

本実験の結果から、椅子型空中像インタフェースにおける空中像の飛び出し距離として、**0～15 cm**が強く推奨される。すべての評価指標（タスクパフォーマンス、主観評価、認知負荷）において、この範囲で最良の結果が得られた。特に**15 cm**は、読みやすさが高く（中央値5.0）、パフォーマンスが良好で（中央値22.5秒）、ワークロードが最も低い（中央値35.7）というバランスの良い距離である。また、0 cmと比較して顔への圧迫感がないという利点もある。

一方、**60 cm**条件では、すべての指標で最も悪い結果となった。読みやすさは著しく低下し（中央値2.0）、遂行時間は約22%増加し（27.4秒）、ワークロードは約60%増加した（中央値56.2）。したがって、45 cm以遠の距離は実用上避けるべきである。

実装においては、デフォルトの飛び出し距離を**15 cm**に設定し、ユーザーの好みに応じて0～30 cmの範囲で調整可能にすることが望ましい。また、やむを得ず遠距離を使用する場合は、文字サイズを視距離に連動して拡大する補償機構の導入が不可欠である。

多言語コミュニケーション支援という本研究の最終目標に照らせば、視認性の確保がユーザー体験の質を大きく左右することが実証された。実際の多言語対話場面では、より長い文章や連続的な情報提示が想定されるため、近距離配置による高い視認性の確保が実用上必須である。

### 4.4.4 制限事項と今後の課題

本研究にはいくつかの制限事項がある。第一に、サンプルサイズが比較的小さい（*n* = 8）。今回は3つの主要指標すべてで有意な効果が検出されたが、より大規模なサンプル（*n* ≥ 15）での追試により、結果の一般化可能性をさらに高めることができる。特に、事後検定での個別比較において、サンプルサイズの増加により有意差が検出される可能性がある。

第二に、実験時間が短い（各条件約1.5分）ため、長期使用における疲労の蓄積効果を評価できていない。実際の使用場面を想定すると、より長時間（例: 30分以上）の連続使用実験が必要である。短時間実験で観察された距離効果は、長時間使用ではより顕著になる可能性がある。

第三に、実験環境の生態学的妥当性が限定的である。本実験ではマネキンヘッドを対話相手として使用したが、実際の多言語コミュニケーション場面では、相手の表情や動き、視線といった社会的要因も影響する。より現実的な対話場面での検証が求められる。

第四に、タスクが比較的単純である。4文字の読み取りは実際の会話における情報処理より単純であり、より複雑な文章（例: 20文字以上の文）や長時間の連続的な情報処理では、距離効果がより強く現れる可能性がある。

今後の研究では、これらの制限を克服するとともに、客観的指標（眼球運動、調節機能、瞬目回数、瞳孔径など）の併用、NASA-TLXの下位尺度の詳細分析（どの要素が距離の影響を受けやすいか）、および個人差の要因（年齢、視力、調節能力など）の検討が望まれる。また、年齢層による違い（特に調節力が低下する40代以降）の検証も重要である。

## 統計分析に関する補足説明（Methods セクション用）

### 統計解析

すべての統計解析はPython 3.10（pingouin 0.5.5、scipy 1.15.3）を用いて実施した。正規性の検定にはShapiro-Wilk検定を用い、有意水準は5%とした。

**タスクパフォーマンス**および**主観評価**については、一部の条件で正規性が満たされなかったため、ノンパラメトリック手法であるFriedman検定を採用した。Friedman検定は反復測定ANOVAのノンパラメトリック版であり、データを順位に変換して検定を行う。

**認知負荷（NASA-TLX）**については、すべての条件でShapiro-Wilk検定により正規性が確認されたため、Friedman検定に加えて、パラメトリック検定である反復測定ANOVAも実施した。ANOVAでは、Mauchlyの球面性検定を行い、球面性の仮定が満たされない場合はGreenhouse-Geisser補正を適用した。事後検定には対応のあるt検定（paired t-test）を用い、多重比較補正にはHolm-Bonferroni法を適用した。

効果量として、Friedman検定にはKendallの一致係数（*W*）を、ANOVAには偏イータ二乗（partial η²）を、t検定にはHedges' *g* を算出した。Kendall's *W* は、0.1未満を効果なし、0.1以上0.3未満を小さい効果、0.3以上0.5未満を中程度の効果、0.5以上を大きい効果と解釈した。

Friedman検定で有意差が認められた場合（*p* < .05）、事後検定としてWilcoxon符号順位検定をすべてのペアで実施し、多重比較補正にはHolm-Bonferroni法を適用した。主観評価では6つの評価項目を同時に検定するため、Bonferroni補正（α = .05 / 6 = .0083）による多重検定の調整も行った。

記述統計量として、中央値、平均値、標準偏差を算出した。リッカート尺度データおよび小標本データでは中央値を主要な代表値として報告した。
