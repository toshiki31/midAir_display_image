## 利用手順

**Activate(Linux, Mac)**

```bash
source [venvname]/bin/activate
```

**パッケージのインストール**

```bash
([venvname])$ pip install [package name]
```

**Deactivate**

```bash
([venvname])$ deactivate
```

**実行**

```bash
python [filename].py
```

## 新しい環境の作成

```bash
cd [project dir]
python3 -m venv [venvname]
```
